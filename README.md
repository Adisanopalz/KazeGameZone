### ✨ Interactive Gaming Web Portal Prototype
KazeGameZone adalah web application modern yang dirancang untuk para gamer agar dapat menemukan game gratis terbaik dan memantau diskon game PC secara real-time dalam satu platform elegan.

Dibangun menggunakan **Vanilla JavaScript**, **Tailwind CSS**, dan desain **Glassmorphism UI** dengan animasi yang smooth serta pengalaman pengguna yang responsif di berbagai perangkat.

# 🌐 Public APIs Used

## 🎮 FreeToGame API

Digunakan untuk mengambil data game Free-To-Play secara real-time.

### ✨ Includes
- Game title
- Thumbnail
- Genre
- Platform
- Release date
- Publisher & Developer
- Official game link

### 🛡️ Extra System
Project ini menggunakan **Triple Fallback Proxy System** untuk mengatasi masalah CORS browser:

- AllOrigins Proxy
- CodeTabs Proxy
- Direct API Fallback

---

## 🏷️ CheapShark API

Digunakan untuk mengambil data diskon game PC secara live.

### ✨ Includes
- Sale price
- Normal price
- Discount percentage
- Steam deals
- Metacritic score
